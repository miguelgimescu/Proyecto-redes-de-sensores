# Proyecto-redes-de-sensores

Este proyecto tiene como objetivo desarrollar un sistema capaz de detectar y clasificar distintos tipos de golpes realizados con una pala de ping pong.

La idea principal es identificar automáticamente el tipo de movimiento o impacto (por ejemplo, golpes planos, cortados, reves, etc.) mediante el análisis de señales capturadas durante el juego.

##  Objetivos del proyecto

- Detectar golpes realizados con una pala de ping pong mediante sensores de movimiento.
- Clasificar diferentes tipos de golpes según su patrón característico.
